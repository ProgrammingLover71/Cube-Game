# Cube Game

Cube Game is a (intentionally) low-quality remake of Geometry Dash, the popular game where you:

    Open the Game ---> Play ---> Rage Quit ---> Cry ---> Repeat

It's made in Python, so you won't have to worry about builds or portability or all that stuff - You can play on *any* PC!

# But *why*?

I created Cube Game because, honestly, I was too lazy to buy GD. Not only that, but I want to give it some features that vanilla Geometry Dash doesn't have, like what you'll read in about 5 seconds (unless you don't care I guess).

# Features

Cube Game has lots of features the GD has, such as:

- Main Levels (7 of them)
    * The main levels in Cube Game are different, so you will actually have something new to play off the bat!
    * They're also harder! I never said it was for beginners.
- Practice mode!
- Lots of Icons!
- Quests and Rewards!
- *Secrets...*
- Level Editing!
- Publishing Levels!
    * *Note:* Level publishing will for now only be available on local servers, so you can share your levels with your close friends. Don't expect a 24/7 global server too soon...

Not only that, but this silly little game also has features that you *can't* find in vanilla GD, like:

- Built-in support for any song you like. Literally any song. Just drop the file in and it works!
- Improved menus! <small><small>(weren't you bored of the old ones...?)</small></small>
- Polished editor!
- *Infinite Editor Area!* You heard me!

And more...

# Installation

Installing Cube Game is easy! Just download it from Github ([here](https://github.com/ProgrammingLover71/CubeGame)) or use:

```bash
python -m pip install cubegame
```

You might be wondering why Cube Game even *is* on PyPI. It's because it has an API that allows you to programatically open the game, then send inputs into it from your script. Why does that exist? *I don't know!*
